# Bot-framework for Java

간단한 에코봇 생성

## Environments

`JDK - openjdk version "11.0.7"`

`Build Tool - gradle`

## Prerequisites

봇 생성후 id 와 패스워드 발급.

테스트 환경을 위한 `ngrok` 터널링
